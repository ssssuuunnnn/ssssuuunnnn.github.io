大概簡單的嘗試了這個套件，我覺得最大的優點就是nginx的管理上加上了權限及Log的作法，也因此變的更加好管控，因為以前如果沒有做好Server上帳號的管控，就會發生有人改了config檔確沒有人知道誰改，為什麼Server突然壞掉的問題。這個套件就可以很好的解決這個問題。

第二，因為這個套件提供了UI介面讓使用者可以操作，所以在對於nginx的新手就會變的相對友善，可以在介面上點一點即可，入門也相對簡單。

第三，套件中直接內建 Let's Encrypt 的服務，不用像以前還需要在 Server 上安裝其他套件，在以指定的方式做操作，對工程人員來說，可能可以加快管理的效率。

不過在這麼多好處之下，我認為還是有一些缺點，最大的問題大概就是資安問題，原因是因為此套件不是原生官方所提供，是熱血的網友自製的，所以在詳細的運作方式可能有什麼風險無法確定。因此，如果是自己的小專案我想是非常適合，但是如果是在防火墻有做好相關的設定，或許就可以解決這個缺點。

今天就分享到這邊，謝謝大家。
